Project Overview:

BlueJobs was built using Python, Django, HTML, CSS, and JavaScript. The project is hosted on GitHub with the link https://github.com/nicsdv/bluejobs. All progress and necessary resources are documented in this repository, with instructions on how to run the repository provided in the README.md file. This project was created in fulfillment of the requirements of CSCI 42: Introduction to Software Engineering. It serves as a practical application of software engineering concepts and principles learned during the course.
